1. Create a folder dataset in a location where all the codes were placed.
2. Firstly, open dataset.py code in python3 idle and click on run and then run module. Then give an id number and enter.
3. All captured samples will be stored in dataset folder.
4. Secondly, open training.py code in python3 idle and click on run and then run module.
4. With machine learning ability, the code will train the raspberry pi with the sample images from the folder by creating a trainer.yml file.
5. Finally, open and run the recognition.py code the raspberry pi will able to detect the face of person with its id.
